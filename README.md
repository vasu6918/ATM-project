# ATM Banking System

A simple Python-based ATM Management System developed using functions, input validation, and multiple account support.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Multiple account support
- 4-digit PIN authentication
- Input validation using try-except
- Menu-driven interface

## Technologies Used

- Python

## Concepts Used

- Functions
- Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- User Input Validation

## Sample Accounts

| Account Number | PIN  | Balance |
|----------------|------|----------|
| 1001           | 1234 | 5000     |
| 1002           | 5678 | 10000    |

## How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

```bash
python atm.py
```
