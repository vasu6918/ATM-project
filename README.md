# ATM Banking System

A simple Python-based ATM Management System developed using functions, input validation, and multiple account support.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Multiple account support
- 4-digit PIN authentication
- Input validation using try-except
- Menu-driven interface

## Technologies Used

- Python

## Concepts Used

- Functions
- Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- User Input Validation
